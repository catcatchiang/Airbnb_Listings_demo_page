# Airbnb Listings Project

## Overview
This project displays a list of Airbnb listings using a JSON dataset. The page dynamically loads the data and shows each listing in a Bootstrap card, including:

- Listing title
- Listing image
- Price
- Host name and host picture
- Amenities

## Technologies Used
- HTML
- CSS
- JavaScript (fetch API)
- Bootstrap 5
- JSON data
